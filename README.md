# 仙途 · Immortal Path (v2.0)

A mobile-first, text-based Wuxia Life Simulator built with HTML5, CSS3, and Vanilla JavaScript. 

Inspired by the "BitLife" style of gameplay, this project explores the journey of a cultivation clan across multiple generations in a world of Chinese fantasy.

## 🌟 Core Features
- **Dynasty & Lineage:** Play through generations. Your current character's legacy (Qi and Wealth) influences the potential of your successors.
- **Sect System:** Join various sects (Azure Peak, Crimson Blade, Thousand Jade Palace) to gain unique techniques and bonuses.
- **Occupation Tree:** Advance from a simple Spirit Farmer to a legendary Pill Sage or Demon Slayer.
- **Spiritual Root System:** Your elemental affinity determines your cultivation difficulty—from the gifted Heavenly Pure Root to the challenging Chaotic Root.
- **Manuals & Knowledge:** Build a clan library of ancient scriptures to provide permanent buffs to your bloodline.

## 🛠️ Technical Stack
- **Language:** JavaScript (ES6+)
- **Styling:** CSS3 (Custom Design Tokens, Mobile-Responsive Layout)
- **Structure:** Clean HTML5 Semantic tags
- **Deployment:** Optimized for conversion to Android APK via Capacitor/PWABuilder

## 🚀 How to Play
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Switch to "Inspect Element" -> "Mobile View" for the best experience.
